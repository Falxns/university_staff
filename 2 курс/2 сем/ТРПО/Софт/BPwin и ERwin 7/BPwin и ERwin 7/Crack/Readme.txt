Инструкция по грамотной установке AllFusion ERwin Data Modeler r7 b1802 и AllFusion Process Modeler r7 b1247.
1. Удалите установленные продукты версии 4.
2. Удалите CA Register IT. Для этого распакуйте куда-нибудь архив C:\CA_LIC\lic98_uninstaller.zip и запустите rmlicense.bat.
3. Установите программу.
4. Найдите, куда был установлен CA Register IT. Путь можно посмотреть в  HKEY_LOCAL_MACHINE\SOFTWARE\ComputerAssociates\License\Install Path, по умолчанию - C:\Program Files\CA\SharedComponents\CA_LIC\
5. Скопируйте файлы Product.dat и RegIT.exe в эту папку.
6. Запустите RegIT.exe.
7. Введите любой e-mail.
8. Выберите Get Customer ID By Phone.
9. Выберите любую страну. Введите любые 3 символа в Zip/Postal Code. Введите любые 8 цифр в Customer ID.
10. запустите кейген от SlaSk/PFT (Keygen.exe). Выберите продукт. Нажмите Generate ID и скопируйте Registration ID.
11. Выберите продукт в Register IT. Вставьте Registration ID. Нажмите Create Registration Code.
12. Скопируйте первую половину Registration Code в кейген. Нажмите Generate Keycode.
13. Скопируйте Keycode в Register IT и нажмите Install. Если всё нормально, то появится сообщение "RegisterIT has successfully installed the keycode.".
14. Повторите шаги 10-12 для второй половины Registration Code.
15. Закройте Register IT (Cancel) и кейген.
Чтобы не повторять всю эту процедуру после переусановки системы, можете сохранить файл ca.olf из папки CA_LIC и восстанавливать его после установки. Скорей всего (не проверял) на разных машинах не будет работать.
